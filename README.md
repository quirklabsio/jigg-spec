A `.jigg` file is a portable digital jigsaw puzzle container format combining an immutable puzzle definition with optional user progress state, packaged as a ZIP archive. The puzzle definition (image, piece geometry, solved positions) is strictly separated from user state, so a `.jigg` without state is a fully valid, shareable puzzle that anyone can open and solve.
